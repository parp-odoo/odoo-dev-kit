# Odoo Dev Kit - The VsCode Extension

### Odoo Server Management
- Configure and manage multiple `addons_path` profiles
- Start Odoo server with custom options:
    - Database name
    - Port
    - Install/upgrade specific modules
    - Custom config file
- One-click:
    - Run tests
    - Module upgrade
- Debug mode support
- Auto-detect Odoo version from workspace 

### Smart Branch & Version Control:
- One-click branch checkout
- Create new branch from current branch
- Auto switch working database based on branch
- Auto-update Odoo version when switching branch
- Sync additional addons repos when version changes
- Quick rebase workflow (interactive option)
- Push with upstream auto-set
- Branch-to-database mapping system
- Warn if DB schema mismatch detected
